​​诞生之1986年，今年迎来35周年的《密特罗德》系列，至今为止经历了无数进化，一直受到众多粉丝的喜爱。在2021年10月8日即将发售的Nintendo Switch最新作《密特罗德 生存恐惧》中，出现了“E.M.M.I.”的新威胁，在以“恐惧”为主题的本作中，本系列将获得比以往更大的进化。

继《密特罗德萨姆斯归来》后，由任天堂和西班牙游戏工作室水银蒸汽共同开发的新《密特罗德》，其诞生背后有着怎样的辛苦呢。为此，我们采访了自《密特罗德》系列诞生以来就一直深入参与其中的制作人坂本賀勇先生，他对《密特罗德》有着怎样的注重呢。

坂本賀勇氏（さかもと よしお）
从初代《密特罗德》诞生至今35年，一直参与系列的制作，继《萨姆斯归来》后，本作也担任制作人。

### 描写毅然面对恐惧的萨姆斯的身姿

##### 恭喜《密特罗德》系列迎来35周年，请告诉我您现在的心情。
{: .question }

##### 坂本
35年时间之长，我能够坚持到现在，是粉丝们和周围的人们都支持着萨姆斯的缘故，真是感慨万千。

##### 最新作《生存恐惧》也终于要发售了。
{: .question }

##### 坂本
实际上，这个构想本身已有15年了，但我对满足我期望形式的质量感到非常满意。

##### 《生存恐惧》与15年前的构想相比有什么变化吗？
{: .question }

##### 坂本
E.M.M.I.是15年前开始构思的概念，不过当时所设想的只有核心部分，故事和游戏设计都还在摸索中。

##### 将构想落实到游戏上，共同开发了《萨姆斯归来》的水银蒸汽团队是否也发挥了很大作用呢？
{: .question }

##### 坂本
是的，他们的技术力和感知力都很棒，并且最重要的，他们也非常珍惜《密特罗德》这个游戏。再加上我们不是像以前那样“按原样”重制《密特罗德》，而是提出新想法新建议，我们能够作为一个团队，在《萨姆斯归来》后继续共同开发真是太好了。

上图为于2017年发售的Nintendo 3DS游戏《密特罗德2 萨姆斯的回归》的重制作《萨姆斯归来》。新动作“近战反击”和“艾翁能力”在《生存恐惧》中继续沿用

##### 我认为恐惧是《密特罗德》系列本身具有的要素之一，请您谈谈这次是为何决定大力强调该要素的。
{: .question }

##### 坂本
从制作初代《密特罗德》开始，我就受到电影《异形》等作品的影响，试图构造恐怖、黑暗的世界观，但并没有强烈意识到恐怖是主题，仅仅觉得，玩家们会从那样的世界观中和无法轻易战胜的强大敌人上感受到恐惧吧。在此基础上，本次游戏我们想重点描绘萨姆斯所感知的恐惧。

在《生存恐惧》中，萨姆斯面对绝望，感到恐惧，但她毫不畏惧，勇敢地开辟了道路，我们想要展现这种积极的态度。

### 从多个角度表现萨姆斯的恐惧

#####  在描绘成为本作重要要素的恐惧的基础上，特别注重的是？
{: .question }

##### 坂本
从大的方面来说，是E.M.M.I.的动作，一种无机物的存在为了抓住你而淡然逼近的恐怖。为了表达这一点，我向水银蒸汽团队解释了很多，展示了不少类似我想象中的E.M.M.I.动作的比如哑剧视频啊各种各种的东西。

我一直想在创作新游戏的时候加入挑战自我，这次《生存恐惧》中加入了此前没有的E.M.M.I.的存在，所以是特别大的挑战。 我希望我们能够将粉丝熟悉的《密特罗德》和新元素平衡地融合在一起来建立一个新的《密特罗德》。

##### 15年前的构想里E.M.M.I.已经是现在的设计了吗？
{: .question }

##### 坂本
设计虽然不同，但当时就有了让人形机器人以四足行走的方式追赶萨姆斯的构想。在怪谈等的演出中也常有，一个像人一样形态的存在以四足步行的方式追赶的桥段，我想很多人也会感到生理上的恐惧。

在《生存恐惧》中，E.M.M.I.的追赶动作也表现出了恐怖，在追赶萨姆斯的时候，它双脚行走，慢慢地靠近萨姆斯，不同的场景，使用了不同的动作。

##### E.M.M.I.为了探测萨姆斯而发出的声音也令人毛骨悚然印象深刻。这个有特色的声音是开发初期就和E.M.M.I.一起考虑的吗?
{: .question }

##### 坂本
是的，实际上这个声音是我小时候听到的斑鸠的鸣声。从我目前听到的声音中突然想到，“我在正常情况下不会害怕听到它，但在特殊情况下我觉得它听起来很可怕。”

##### 相信很多玩家在看过预告后都对E.M.M.I.压倒性的强大感到震惊。
{: .question }

##### 坂本
“E.M.M.I.区域”是一个恐怖的区域，我们想把它和一般的可以自由探索的地方区别开来。为了表现萨姆斯视角的恐惧，会有像水蒸气一样的东西弥漫在画面中，并加入像噪音一样的特效，视觉要素也是很注重的。

只是，根据游戏的进度，萨姆斯也会掌握对抗E.M.M.I.的能力，所以通过力量的提升从而得以与之对抗，这样的设计与其他敌人一脉相承。

##### 为了打倒敌人而摆出欧米伽加农炮时的“锁定聚焦模式”，虽然是2D动作游戏却像TPS一样，这挺新鲜的。
{: .question }

在锁定聚焦模式模式下，画面会更接近萨姆斯，并且可以看得比平时更远

##### 坂本
《密特罗德》是一款2D卷轴动作游戏，玩家可以灵活地移动萨姆斯。只是，在描写E.M.M.I.的恐怖上，我觉得普通视角的画面可能不够。通过采用接近主观的视角，可以明确地感受到E.M.M.I.从一般画面所不展示的遥远地方逼近的模样，也因为看不到背后的情况，所以会有恐惧。

既兼顾演出效果有不落游戏体验，这种理想形式费了我不少功夫，但最后还是顺利地成型了。

### 两家公司联手打造的新《密特罗德》

##### 在动作方面也追加了冲刺近战等要素，这些新要素是怎样诞生的呢？
{: .question }

##### 坂本
近战动作最初是水银蒸汽团队提出的想法。虽然在《萨姆斯归来》中便加入了新动作，但我们认为在本作里要更加积极，不要干扰游戏的流程才行。

因此，任天堂方提出了追加冲刺近战的提案，并与水银蒸汽团队合作，将其落实到游戏中。在其他方面，我们也互相提出了自己的想法。

##### 水银蒸汽团队里有很多喜欢《密特罗德》的工作人员，提案的想法是和玩家的角度相近的比较多吗？
{: .question }

##### 坂本
这方面自然是有的。当然，我们也会从玩家的角度考虑问题，但能获得水银蒸汽团队的角度考虑的问题，我还是非常感激的。包括总监José先生在内，水银蒸汽团队里有太多喜欢《密特罗德》的人，所以他们很认真地思考如何改进《密特罗德》。

##### 受到追加的动作的影响，游戏的节奏也比之前快了很多。这是坂本先生最开始的目标吗？
{: .question }

##### 坂本
至今为止我已经制作了很多《密特罗德》，但是我认为每个人都有适合的节奏，在这次的《密特罗德》中，迅速的动作是最适合的调整。

虽然制作了符合各个《密特罗德》作品设计的游戏动作，但这次的动作非常舒适且易于操作，我自己也很喜欢。对于这次的《密特罗德》来说，可能是最好的动作。

##### 迅速且易上手的动作得以实装，受这些追加动作的影响，难易度的调整也变得和之前不同了吧?
{: .question }

##### 坂本
任何作品的难易度都很难调整，我们在开发过程中习惯了难易度，有时会难以理解第一次游玩的人的感受，通过由任天堂和水银蒸汽一同操手难易度调整，我们能在注重于它的同时顺利进行。

但是，由于E.M.M.I.的要素还没有人挑战过，所以我们请了很多人实际游玩，以将其调整到动作游戏的最佳平衡，其中最重视的是失败游戏结束时，能否让玩家产生“下次我要这样做”的想法。

### 续关丝滑，可以从被打倒的地方附近重新开始

##### 是不是形成了玩家想要复仇的平衡呢。
{: .question }

##### 坂本
比如说，不怎么会被E.M.M.I.抓住，或即使被抓住也能马上逃跑的话，游戏的根本要素——E.M.M.I.的恐怖就不能充分表现。话虽如此，因为太难也会造成压力，所以E.M.M.I.的平衡调整很辛苦。

##### 此外，萨姆斯的power suits的设计也以一种易于理解和新颖的方式进行了改动，您在设计时意识到了哪些方面？
{: .question }

##### 坂本
因为本作是《密特罗德 融合》的续作，所以我想一边保留着“fusion suits”的印象，一边将有机和无机的东西融合在一起进行设计。设计上我们也和水银蒸汽方面讨论了很多。

##### 比其以前的power suits，更能感受到女性力的味道。
{: .question }

##### 坂本
确实是和《萨姆斯归来》不同的外型，玩家也有很多这样的意见，但其实我们并没有特别意识到，所以这些反应让我们很意外。

##### power suits设计是很多粉丝关注的要素，整个系列中，您在设计上有什么看重的地方吗？
{: .question }

##### 坂本
萨姆斯的设计会根据世界观进行微调，使视觉效果与每部作品的动作相匹配来体现她的英姿飒爽。另外，关于敌人，我也有意识地加入了怪诞的要素。无论《萨姆斯归来》还是本作，水银蒸汽团队都向我们提出了各种各样的设计方案，我们互相反复征求意见，不断打磨。即使没有直接说“我想要敌人这样做”，我们也能做出符合《密特罗德》风格的设计，真是帮了我们大忙。

##### 从初代《密特罗德》开始玩家们就很熟悉的敌人角色克雷德的设计也很特别。
{: .question }

##### 坂本
虽然一看就知道是克雷德，但是我想让大家看到只在《生存恐惧》才有的新克雷德。在游戏中怎样战斗也和设计的表现方式有关，我们一边缜密地交流一边将形象磨合。

##### 《密特罗德 生存恐惧》是继sfc作之后发售的再次可以在家用机上游玩的2D动作《密特罗德》，可以分享在开发高清游戏时遇到的困难吗？
{: .question }

##### 坂本
因为这是我第一次制作高清版《密特罗德》，所以我费了很大的劲决定在品质到底要追求到什么程度，最后得到了任天堂各个部门的人的协助。这点，水银蒸汽团队是有着丰富的高清游戏开发经验的团队，在《生存恐惧》的开发中，我依赖了很多。

##### 也就说在游戏设计方面并不是特别辛苦。
{: .question }

##### 坂本
硬件的性能之提升，能做的事情也增加了，不如说开发本身就是在舒适的环境中进行的。

### 《密特罗德 生存恐惧》所展示的探索型动作游戏的新可能性

##### 请说说至今为止的系列作品开发的回忆.
{: .question }

##### 坂本
回忆深刻的事情有很多，很难锁定到一个范围。当然，和水银蒸汽团队的相遇也是如此，和Team NINJA一起制作《密特罗德 另一个M》的时光亦是非常充实。

此外，《密特罗德 融合》是开发《瓦力欧》系列的团队首次尝试开发《密特罗德》的作品。我通过写在纸上展示，一个个地给负责关卡设计的人解释《密特罗德》的关卡设计，以便他们理解《密特罗德》是什么这件事也印象颇深。

然后，果然初代《密特罗德》的回忆才是最深刻的呢。我参与开发的时候，明明发售日已经定下，但开发却几乎没有进展。这个团队包括我自己在内，都没有太多的游戏开发经验，但当时的任天堂第一开发部全力以赴，才成功推出这个作品。在开发过程中，我也有过担心自己无法完成的时候，但当开发顺利完成，看到完成的职员表时我的心咯噔，觉得真的太好了。这件事我想我一辈子都不会忘记。

##### 《密特罗德》系列的35年历史充满了坂本先生的回忆。请告诉我们您迄今为止在该系列作品的开发过程中特别注重的是什么。
{: .question }

##### 坂本
刚才我也说过，《密特罗德》的一些东西会让人联想到电影《异形》的。汉斯·鲁道夫·吉格尔※那有机生物却有无机的机械音调的设计也是《密特罗德》的理论。

※汉斯·鲁道夫·吉格尔，瑞士艺术家。因《异形》中的生物设计而闻名。“生物机械”这种生物与机械结合的独特艺术风格给各行各业带来了巨大影响。

不过，这些年来这种情况也在慢慢改变，似乎与水银蒸汽的相遇使我们进入了与以前不同的阶段。通过尊重水银蒸汽那来自海外的不同感受，我觉得能够做出更好的东西，我也改变了对工作的态度。多亏了这些变化，才有顺利进行的感觉。

##### 近年来，《密特罗德》等探索型动作游戏主要流行于独立游戏领域，您如何看待这一趋势？
{: .question }

##### 坂本
一想到我们35年前制作的游戏被作为一种类型，被各种各样的作品解释重构，就觉得非常有趣，非常光荣。如果这一类型还有可能性的话，今后人气也会继续下去，我们想表现出这种可能性，我希望《生存恐惧》能够激发独立游戏创作者们灵感。

##### 挑战类型可能性的全新作《密特罗德 生存恐惧》，最后请对等候这一作发售的粉丝们留言吧。
{: .question }

##### 坂本
恭候完全新作的各位粉丝们，让您久等了非常抱歉。具备了各种条件，终于迎来发售的《密特罗德 生存恐惧》，可以说是迄今为止2D动作游戏《密特罗德》的集大成之作。

E.M.M.I.、谜一样的鸟人族、克雷德等各种各样的要素，这些要素是如何交织在一起，使萨姆斯的故事更加精彩呢，以及密特罗德和萨姆斯的故事又会如何告一段落呢。请一定要实际游玩确认一下。
